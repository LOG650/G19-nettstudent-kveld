# 03 Dokumentere datasett (WBS 2.4)

Denne aktiviteten dokumenterer datasettet i en tabellprofil og en kort markdown-oppsummering.

## Kjøring

Kjør fra roten av `006 analysis`:

```bash
uv run python aktiviteter/03_dokumentere_datasett/start_wbs_2_4.py
```

## Filer i mappen

- `start_wbs_2_4.py`: Genererer datasettprofil.
- `tab_dataset_dokumentasjon.csv`: Kolonneprofil med datatype, manglende verdier, unike verdier, min/max og eksempelverdier.
- `dataset_dokumentasjon.md`: Kort tekstlig oppsummering for rapportgrunnlag.

## Datakilde

- `004 data/Dagligvare_Dataset.csv`
